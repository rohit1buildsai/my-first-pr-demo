# my-first-pr-demo

Demo repo for my first PR.

## About

This repository was created to practice the full GitHub pull request workflow — from cloning a repo to opening and merging a PR.

## Getting started

1. Clone the repo: `git clone https://github.com/rohit1buildsai/my-first-pr-demo`
2. Create a branch: `git checkout -b your-branch-name`
3. Make a change, commit, and push
4. Open a pull request on GitHub
